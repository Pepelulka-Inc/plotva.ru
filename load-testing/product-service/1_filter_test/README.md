# Сценарий для нагрузочного тестирования с запросами фильтрации.

Запуск нагрузочного тестирования:

1. Запустить БД и сервис product-service.
2. Создать папку `data` и файлы в ней `cats.json` и `sellers.json` 
3. Ввести нужные значения в скрипте gen.py и запустить его. Он сгенерирует нужные категории и продавцов, запишет
их в файлы `data/{cats,sellers}.json`. Так же он сгенерирует рандомные товары и загрузит их в БД.
4. Ввести нужные значения в скрипте `test.js` и запустить нагрузочный тест командой `k6 run test.js`
